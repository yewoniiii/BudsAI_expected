Expected Effects 01

사용성 개선 효과: “방금 들은 소리”를 바로 이해하고 실행으로 연결

BudsAI는 사용자가 놓친 청각 정보를 다시 찾고, 답변을 넘어 다음 행동까지 이어주는 경험을 제공한다.
기존 음성비서가 “사용자의 발화”만 이해했다면,
BudsAI는 직전 청각 맥락 + 현재 질문을 함께 이해한다. 사용자는 앱을 열거나 검색어를 직접 입력하지 않아도, 버즈에서 바로 질문하고 결과를 받을 수 있음
Hands-free · Context-aware
↺

놓친 정보 회복

“방금 뭐라고 했어?”, “이 노래 뭐야?”처럼 순간적으로 지나간 소리도 직전 버퍼 기반으로 다시 확인할 수 있다.

⚡

탐색 단계 단축

음악 검색, 번역, 요약, 일정 추출을 별도 앱 전환 없이 처리해 사용자의 조작 부담과 탐색 시간을 줄인다.

✓

답변에서 실행까지

Galaxy에서는 분석 결과를 Action Item으로 제공해 음악 열기, 일정 등록, 알림 설정 등 후속 행동으로 연결한다.

Expected Effects 02

당사 제품군 확대 적용 효과

BudsAI를 Buds 단일 기능이 아닌 Galaxy 생태계 확장형 Audio AI 경험으로 적용
B

Buds 라인업 가치 상승

Buds3/Buds Pro 후속 모델의 핵심 차별화 기능으로 탑재 가능 → 단순 무선 이어폰을 "AI 청취 어시스턴트"로 포지셔닝 전환.

G

Galaxy 생태계 락인 강화

Galaxy 폰·워치·태블릿·TV·차량(One UI Auto)에 컴패니언 레이어 확장 시 "삼성을 쓰면 BudsAI가 더 똑똑하게 동작" 차별점 확보 → Galaxy 구매 동인.

S

Bixby·SmartThings 재활성화

기존 자산(Bixby NLU, SmartThings, Wearable) 재사용 → 신규 인프라 투자 최소.

↔

타사 호환을 통한 저변 확대

비-Galaxy 호스트도 동작 → 진입 마찰 0, 호환 버즈 확산 시 즉시 활용. 이후 Galaxy 특화 UX로 자연스러운 업셀.

＋

새 호환 기기 즉시 적용

표준 BT PAN 기반이라 향후 출시되는 갤럭시 차량·TV·웨어러블 등 BT+인터넷 가능한 모든 신제품에 추가 개발 없이 적용 가능.

D

데이터 자산화

(사용자 동의 하) 음성 의도 분류·환경음 인식 데이터 축적 → 향후 온디바이스 AI 모델 학습 자원.
