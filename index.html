Expected Effects 01

사용성 개선 효과: 듣고 있던 순간을 바로 질문하고 실행으로 연결

BudsAI는 “기억해서 검색하는 경험”을 “방금 들은 맥락에 바로 묻는 경험”으로 바꾼다.
기존 사용 흐름
소리 인지앱 열기검색어 추정검색/전환결과 확인
→
BudsAI 사용 흐름
롱프레스자연어 질문직전 사운드 분석답변 + Action
↺

놓친 정보 회복

지나가 버린 안내방송, 대화, 음악을 직전 버퍼 기반으로 다시 확인해 청각 정보의 일회성을 줄인다.

◎

맥락형 질의

“이 노래”, “방금 그 말”, “이 소리”처럼 검색어로 바꾸기 어려운 표현도 Track A와 결합해 해석한다.

↯

조작 부담 감소

이동 중·운동 중·양손 사용 중에도 화면 전환 없이 버즈 interaction과 음성만으로 결과를 받을 수 있다.

✓

답변 이후 실행

Galaxy에서는 음악 열기, 일정 등록, 알림 설정 등 후속 Action Item으로 이어져 탐색 후 행동 단계를 줄인다.

사용성 핵심 가치: 기억·검색·전환 비용 감소 → 즉시 이해·저장·실행 경험 제공 Listen → Ask → Act
Expected Effects 02

당사 제품군 확대 적용 효과

BudsAI를 Buds 단일 기능이 아닌 Galaxy 생태계 확장형 Audio AI 경험으로 적용
B

Buds 라인업 가치 상승

Buds3/Buds Pro 후속 모델의 핵심 차별화 기능으로 탑재 가능 → 단순 무선 이어폰을 "AI 청취 어시스턴트"로 포지셔닝 전환.

G

Galaxy 생태계 락인 강화

Galaxy 폰·워치·태블릿·TV·차량(One UI Auto)에 컴패니언 레이어 확장 시 "삼성을 쓰면 BudsAI가 더 똑똑하게 동작" 차별점 확보 → Galaxy 구매 동인.

S

Bixby·SmartThings 재활성화

기존 자산(Bixby NLU, SmartThings, Wearable) 재사용 → 신규 인프라 투자 최소.

↔

타사 호환을 통한 저변 확대

비-Galaxy 호스트도 동작 → 진입 마찰 0, 호환 버즈 확산 시 즉시 활용. 이후 Galaxy 특화 UX로 자연스러운 업셀.

＋

새 호환 기기 즉시 적용

표준 BT PAN 기반이라 향후 출시되는 갤럭시 차량·TV·웨어러블 등 BT+인터넷 가능한 모든 신제품에 추가 개발 없이 적용 가능.

D

데이터 자산화

(사용자 동의 하) 음성 의도 분류·환경음 인식 데이터 축적 → 향후 온디바이스 AI 모델 학습 자원.
